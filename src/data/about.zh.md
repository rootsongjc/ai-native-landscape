# 关于 AI Native Landscape

AI Native Landscape 是一个面向 AI Native 技术栈的开源项目生态地图，重点覆盖工具、Agent、运行时、平台与基础设施，通过结构化分类、双语描述和透明评分，帮助开发者理解快速演进的技术生态。

## 这个站点关注什么

- Agent 系统、MCP、上下文工程、RAG、AI 运行时与 AI 基础设施
- 具备可验证 GitHub 仓库与基础元数据的开源项目
- 长期维护的策展式收录，而不是一次性聚合

## 这个站点不收录什么

- 闭源产品
- 仅有模型本身、没有项目形态的条目
- 纯学术论文类内容

## 关于 Jimmy Song

Jimmy Song 长期关注 AI 原生基础设施、云原生应用架构与开源生态实践。

他目前担任[密瓜智能](https://dynamia.ai/)开源生态副总裁，同时也是 CNCF Ambassador、云原生社区（中国）创始人。

他长期投入的方向包括 AI 原生基础设施、GPU 虚拟化、异构算力调度、云原生架构演进，以及开源生态协作。

AI Native Landscape 延续的正是这种长期主义的策展方式：强调结构化信息、明确分类和可验证的工程信号，而不是追逐短期热点。

## 为什么要做这个项目

这个项目的目标是帮助开发者快速理解一个项目在 AI 原生技术栈中的位置，比较相邻工具，并发现真正值得持续关注的开源项目。

## 为什么值得信任

- 仅收录 GitHub 上可验证的开源仓库
- 分类与描述均长期维护中英双语版本
- 评分方法公开，尽量保持可理解和可解释
- 项目目录会持续维护，而不是静态罗列
