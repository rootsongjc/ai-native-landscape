## 简介

Claude-Flow 是一个面向开发者的 AI 编排平台，提供 hive-mind 与 swarm 两种多智能体运行模式，支持实时控制、Hooks 自动化、以及与 Flow Nexus 云平台的深度集成。

## 主要特性

- 高并发 agent 管理与会话分叉，显著提升并行效率。
- 丰富的 MCP 工具集与 Hook 系统，支持自动化工作流与权限分层。
- 原生 Claude Code SDK 集成与 Flow Nexus 云沙箱部署支持。

## 使用场景

- 构建复杂多步骤的开发自动化、代码生成与系统集成流水线。
- 在云端或本地运行多智能体协作任务，支持实时暂停/恢复/终止控制。
- 研究与基准测试 agent 协同策略与性能优化。

## 技术特点

- 使用 TypeScript/Node.js 为主的实现，提供命令行工具和 SDK，丰富的文档与示例。
- MIT 许可证开源，社区活跃，提供 Wiki 和 Flow Nexus 在线文档。
