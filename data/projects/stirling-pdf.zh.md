## 简介

Stirling PDF 是一个开源、可本地部署的网页端 PDF 编辑与处理平台，提供超过 50 种 PDF 操作，包括合并、拆分、转换、OCR、压缩与页面编辑，适用于个人与企业场景。

## 主要特性

- 丰富的 PDF 操作：合并、拆分、旋转、裁剪、压缩、OCR、格式转换等。
- 并行处理与自定义流水线，支持批量任务与下载选项。
- 本地托管与 Docker 支持，关注隐私与企业部署（支持 SSO、备份与数据库集成）。

## 使用场景

- 企业与组织的文档处理自动化与批量转换工作流。
- 本地化合规要求下的 PDF 编辑与审阅，避免将文件上传至第三方服务。
- 与外部脚本或 API 集成，用作后台文档处理微服务。

## 技术特点

- 基于 Java 与前端组件，使用 Docker 容器化部署以便快速上线。
- 支持 LibreOffice、Tesseract OCR 等工具完成格式转换和识别任务。
- 模块化设计与详细文档（docs.stirlingpdf.com），便于扩展与二次开发。
