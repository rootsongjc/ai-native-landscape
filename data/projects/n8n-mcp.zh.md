## 简介

n8n-MCP 是一款为 n8n 工作流自动化平台和 AI 助手（如 Claude、Cursor、VS Code 等）提供桥接能力的 Model Context Protocol (MCP) 服务器。它让 AI 能够理解、检索和操作 n8n 的 500+ 节点，支持模板优先、批量变更、并行操作和多级校验，极大提升自动化开发效率。

## 主要特性

- 支持 536 个 n8n 节点的文档、属性和操作查询
- 2,500+ 工作流模板智能发现与复用
- 节点配置、连接、表达式等多级强校验
- 并行批量操作与高效变更
- 支持 npx、Docker、Railway 云等多种部署方式
- 完全开源，MIT 协议

## 使用场景

- AI 辅助 n8n 工作流搭建与自动化开发
- 智能体 IDE（如 Claude、Cursor、VS Code）集成
- 企业自动化、数据集成、流程编排
- 快速模板复用与批量工作流管理

## 技术特点

- 高覆盖率节点文档与真实模板示例
- 并行执行与批量变更能力
- 多级校验与错误修复机制
- 低资源占用，支持本地与云端部署
- 完善的 API 与工具链，适配多种 AI 助手
