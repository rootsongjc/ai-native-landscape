## Overview

Olares is an open-source personal cloud operating system built on Kubernetes that lets users reclaim control of their data. It provides a self-hosted platform for running AI agents, serving models locally, and managing personal applications with full privacy.

## Key Features

- Kubernetes-based personal cloud OS with one-click installation
- Built-in local AI model serving and inference capabilities
- Native MCP (Model Context Protocol) support for AI agent integration
- Self-hosted alternative to cloud services with data sovereignty
- Edge AI support for on-device intelligence

## Use Cases

- Run a private AI-powered personal cloud at home
- Self-host AI agents and models without relying on external cloud providers
- Build a homelab with Kubernetes-based AI infrastructure
- Deploy privacy-first AI applications with local data processing

## Technical Details

- Built on Kubernetes with a tailored control plane for personal cloud workloads
- Integrates local LLM serving with AI agent orchestration via MCP protocol
- Supports edge deployment for low-latency on-device AI inference
