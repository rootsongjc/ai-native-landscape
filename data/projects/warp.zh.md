## 简介

Warp 是一个重新定义的终端，作为智能体开发环境构建。它将基于 Rust 的 GPU 加速终端与 AI 能力相结合，提供命令生成、调试和工作流自动化。终端具有现代文本编辑、命令补全和协作工作流等功能。

## 主要特性

- 基于 Rust 构建的 GPU 加速终端，性能卓越
- AI 驱动的命令建议、错误解释和自然语言命令生成
- 现代文本编辑，支持光标定位、选择和多行输入
- 工作流自动化，简化重复命令序列
- 协作分享命令和终端会话

## 使用场景

- 通过 AI 辅助命令生成加速日常终端工作流
- 在终端中直接使用 AI 驱动的错误解释进行调试
- 在团队间共享终端工作流和命令知识

## 技术特点

- 完全使用 Rust 构建，利用 Metal/GPU 渲染 UI
- 使用 WebAssembly (WASM) 实现可扩展性
- 支持 macOS 和 Linux 上的 Bash、Zsh 和 Fish shell
