## Overview

Open Generative AI is an open-source alternative to commercial AI video and image generation platforms. It provides a studio interface with 200+ models including Flux, Midjourney-style image generation, and video creation capabilities, all running locally or self-hosted.

## Key Features

- 200+ AI models for image and video generation.
- Midjourney-style image generation with Flux and other models.
- Video creation studio with multiple generation pipelines.
- Self-hosted with no API costs.

## Use Cases

- Generate images and videos without relying on commercial platforms.
- Run AI generation locally for privacy and cost savings.
- Build custom image and video generation workflows.
