## Overview

Pi Monorepo is a comprehensive AI agent toolkit containing a unified LLM API, coding agent CLI, terminal UI (TUI), web UI components, and deployment tooling to build and run agent-powered applications.

## Key Features

- Multi-package monorepo with reusable SDKs and CLIs.
- Unified adapter layer for multiple LLM providers.
- Developer-first tooling: CLI, TUI, and example apps.

## Use Cases

- Building custom coding agents and developer copilots.
- Embedding AI chat and assistant features in terminal and web apps.
- Managing vLLM deployments and multi-model routing.

## Technical Highlights

- TypeScript-driven monorepo architecture.
- Rich examples and CI tooling for reproducible builds.
