A visualization mcp contains 25+ visual charts using @antvis. Using for chart generation and data analysis.

## Key Features

- **Rich Chart Library**: Contains over 25 visual chart types for diverse data visualization needs
- **AntV Integration**: Built with the powerful AntV visualization library for high-quality chart rendering
- **MCP Compatibility**: Implements the Model Context Protocol for seamless integration with AI agents
- **Data Analysis Ready**: Designed for data analysis workflows and chart generation tasks

## Chart Types

The server includes a wide variety of chart types suitable for different data visualization scenarios:

- Statistical charts (bar, line, pie, area charts)
- Diagrams (flowcharts, network diagrams)
- Advanced visualizations (heatmaps, scatter plots, radar charts)
- And many more...

## Use Cases

- Automated chart generation for reports and dashboards
- Data analysis workflows with AI agents
- Visualization of complex datasets
- Integration with business intelligence tools
