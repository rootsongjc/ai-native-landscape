## 简介

Hyperframes 是 HeyGen 推出的开源工具，将 HTML 转换为渲染视频，专为 AI Agent 设计。它支持从代码程序化创建视频，使 AI Agent 能够大规模生成动态视频内容。

## 主要特性

- 面向 AI Agent 的 HTML 转视频渲染管线。
- 从结构化模板程序化创建视频。
- 设计用于集成到 Agent 工作流和自动化中。
- HeyGen 出品，Apache 2.0 协议。

## 使用场景

- 通过 AI Agent 大规模生成个性化视频。
- 从数据创建动态营销和演示视频。
- 构建自动化视频生产流水线。

- GitHub 23,000+ Star。
- 由 AI 视频领域领导者 HeyGen 构建。
- Agent 优先的 API 设计，实现无缝自动化。
