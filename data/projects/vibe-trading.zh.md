## 简介

Vibe-Trading 是 HKUDS 推出的个人 AI 交易 Agent，结合多 Agent 架构、回测引擎和 MCP 集成，支持算法化交易。一条命令即可为 AI Agent 赋予全面的交易能力。

## 主要特性

- 多 Agent 架构用于交易策略开发。
- 内置回测引擎用于策略验证。
- MCP server 集成实现 Agent 驱动交易。
- 影子账户系统用于无风险测试。

## 使用场景

- 开发和回测 AI 驱动的交易策略。
- 通过 MCP 协议集成运行自动化交易 Agent。
- 在实盘部署前用影子账户测试策略。
