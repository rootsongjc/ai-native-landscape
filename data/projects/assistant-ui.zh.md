## 简介

Assistant UI 是一个基于 TypeScript/React 的开源组件库，用于构建可定制的 AI 聊天界面。它提供可组合的 UI 组件，覆盖消息流、输入区、富媒体渲染和插件扩展点，便于与后端推理服务、模型网关或智能体层集成，帮助团队快速搭建生产级聊天前端。

## 主要特性

Assistant UI 提供组件化的聊天元素（消息列表、输入框、工具栏）并支持自定义渲染器与样式覆盖，支持通过后端网关实现多模型切换与对比，内置插件点用于集成文件上传、富媒体渲染与工具调用，以及开箱即用的主题定制与无障碍最佳实践支持。

## 使用场景

开发者可快速搭建客服、产品助手或内部协作类聊天界面，将不同后端推理服务统一呈现给最终用户，在产品原型阶段验证对话式交互与多模型策略，或作为组件库嵌入低代码平台与企业内网应用以加速落地。

## 技术特点

基于 TypeScript 与 React 开发，具备类型安全与良好扩展性，提供可定制的消息解析与渲染流水线便于实现富媒体与卡片样式展示。项目关注前端性能与构建体积，兼容现代打包与服务端渲染方案，采用 MIT 许可便于二次开发与商业集成。
