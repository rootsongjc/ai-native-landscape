## 简介

MCP Registry 是一个社区驱动的 Model Context Protocol (MCP) 服务器注册服务，致力于为开发者和企业提供 MCP 兼容服务器的发现、发布与管理能力。用户可通过注册中心快速查找、接入和发布各类 MCP 服务，推动 AI 生态的开放与协作。

## 主要特性

- 支持多种认证方式（GitHub OAuth、DNS 验证等），保障服务器归属安全
- 提供 MCP 服务器列表与搜索，方便快速发现和集成
- 开源架构，支持社区贡献与扩展
- 丰富 API 文档与 CLI 工具，便于自动化集成与开发
- 支持多环境部署，兼容主流云平台与本地环境

## 使用场景

- AI 服务开发者发布和管理 MCP 兼容服务器
- 企业或团队快速集成 MCP 服务，实现多模型协同
- 社区用户发现、体验和反馈新型 MCP 服务器

## 技术特点

- 基于 Go 语言开发，架构稳定高效
- 支持 PostgreSQL、Docker 等主流技术栈
- 完善的 API 设计与文档体系，易于二次开发
- 社区驱动，持续迭代与创新
