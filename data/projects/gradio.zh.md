## 简介

Gradio 是一个用 Python 快速创建交互式 AI Web 应用的工具箱，支持在本地、Colab 或 Hugging Face Spaces 一键分享演示。它把复杂的前端交互抽象为简单的组件，适合演示模型、构建原型与教学使用。

## 主要特性

- 几行 Python 即可创建带 UI 的 demo（Interface、Blocks、ChatInterface 等）。
- 支持快速分享（launch(..., share=True)）和托管在 Hugging Face Spaces。
- 丰富的组件库与事件系统，支持多媒体、数据框、绘图等交互。
- 同时提供 Python 与 JavaScript 客户端，便于在不同环境调用 Gradio 应用。

## 使用场景

- 快速为模型或 API 构建可交互的演示页面以供展示与评估。
- 在教学与研讨会中演示模型能力，或者进行用户研究收集反馈。
- 将模型封装为轻量级内部工具或集成到产品原型中。

## 技术特点

- Python-first 生态，兼容多种 ML 后端与浏览器前端技术（Svelte/TypeScript）。
- 提供详细的文档、指南与大量示例（docs、demo 目录），并有活跃社区维护与频繁发布。
- 採用 Apache-2.0 许可，便于商用与社区贡献。
