Open WebUI 是一个功能丰富的自托管 AI 平台，支持完全离线运行。它提供了类似 ChatGPT 的用户体验，集成了多种大语言模型运行时（如 Ollama 和 OpenAI 兼容的 API），并内置了 RAG 推理引擎。

## 主要特性

- 支持 Docker 和 Kubernetes 部署
- 集成 Ollama 和 OpenAI 兼容的 API
- 完整的权限和用户组管理
- 支持移动端和 PWA
- 内置 Markdown 和 LaTeX 支持
- 语音/视频通话功能
- 模型构建器和 Python 函数调用
- RAG 和网页搜索集成
- 图像生成支持
- 多语言界面
- 插件系统支持

项目持续更新维护，定期发布新功能和修复。
