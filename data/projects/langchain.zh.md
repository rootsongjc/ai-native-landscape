## 简介

LangChain 是构建 LLM 应用的领先智能体工程平台。它提供模型、嵌入、向量数据库、检索器和工具等可组合组件，帮助开发团队快速搭建 RAG 管道、智能体工作流及其他生产级 LLM 系统。

## 主要特性

框架内置数十种模型提供商、向量数据库和检索后端的适配器，开箱即用。其组件化架构通过链、智能体和工具的抽象接口，使单个组件的替换和扩展变得简单高效。更广泛的生态系统包括用于可观测性的 LangSmith 和用于有状态智能体编排的 LangGraph。

## 使用场景

团队使用 LangChain 构建检索增强生成（RAG）系统，将 LLM 连接到私有知识库以实现精准问答。它支持多步智能体工作流，编排工具调用、API 集成和推理链。企业还依赖它开发生产级聊天机器人、文档分析管道和自动化数据处理应用。

## 技术特点

LangChain 主要使用 Python 编写，并提供并行的 JavaScript/TypeScript 生态（LangChain.js）以覆盖全栈场景。它支持跨所有主流模型提供商和向量数据库的插件式集成。项目拥有超过十万 GitHub Stars，维护着丰富的文档、教程和企业级示例。
