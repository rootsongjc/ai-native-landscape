## 简介

Build with Claude 是一个集中式资源平台，用于发现 Claude Skills、Agents、Commands、Hooks、Plugins 和 Marketplace 集合，以扩展 Claude Code、Claude Desktop、Agent SDK 和 OpenClaw。它汇集社区与官方扩展，提供可搜索的 Web UI，支持通过 Claude 插件市场或手动仓库方式安装。

## 主要特性

平台提供丰富的插件集合，涵盖智能体、命令、钩子与技能，支持按类别筛选、搜索与一键复制安装命令的插件市场发现功能。包含 MCP 与社区互联索引以连接 MCP 服务器与第三方市场，并提供 Claude Code 插件市场集成与手动安装等多种安装方式。

## 使用场景

适合为团队快速部署领域专家智能体以辅助编码、审计与测试工作，通过命令与钩子自动化常见开发任务。也适用于在安全或受控环境中手动部署精选插件以满足合规与审计要求，以及作为社区发现入口快速找到并试用第三方插件与市场集合。

## 技术特点

基于仓库组织的插件清单与元数据，便于自动化安装与持续更新。Web UI 提供可视化浏览、过滤与示例演示以提高可发现性，支持与 Claude Code 的原生插件机制配合。文档中详细说明插件格式与贡献流程，注重可扩展性与可贡献性。
