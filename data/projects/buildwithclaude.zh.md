## 详细介绍

Build with Claude 是面向 Claude Code 的插件市场与发现平台，汇集智能体、命令、钩子、技能与插件集合，便于用户浏览、搜索并一键安装社区与官方扩展。平台同时提供可导航的 Web UI 与仓库目录结构，支持通过 Claude 插件市场直接添加或按需手动安装，帮助团队快速将智能体与自动化能力集成到工作流中。[访问网站](https://www.buildwithclaude.com)。

## 主要特性

- 丰富的插件集合：涵盖智能体（多语言专家）、命令（例如提交、文档、TDD）与钩子（事件驱动自动化）。
- 插件市场与发现：按类别筛选、搜索与复制安装命令，支持批量安装集合插件。
- MCP 与社区互联：索引 MCP 服务器与社区市场，便于连接数据库、API 与工具链。
- 多平台支持：提供 Claude Code 插件市场集成与手动安装说明，兼容多种工作流。

## 使用场景

- 为团队快速部署领域专家智能体以辅助编码、审计与测试工作。
- 通过命令与钩子自动化常见开发任务（格式化、提交、生成文档等）。
- 在安全或受控环境中手动部署精选插件以满足合规与审计要求。
- 作为社区发现入口，快速找到并试用第三方插件与市场集合。

## 技术特点

- 基于仓库组织的插件清单与元数据，便于自动化安装与持续更新。
- Web UI 提供可视化浏览、过滤与示例演示，提高社区贡献与可发现性。
- 支持与 Claude Code 的原生插件机制配合（marketplace 安装流程）。
- 注重可扩展性与可贡献性，文档中详细说明插件格式与贡献流程。
