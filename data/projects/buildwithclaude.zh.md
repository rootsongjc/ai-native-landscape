Build with Claude 是一个集中式资源平台，用于发现 Claude Skills、Agents、Commands、Hooks、Plugins 和 Marketplace 集合，以扩展 Claude Code、Claude Desktop、Agent SDK 和 OpenClaw。它汇集社区与官方扩展，提供可搜索的 Web UI，支持多种安装方式。

## 插件发现与市场

- 丰富的插件集合，涵盖智能体、命令、钩子与技能，统一可搜索界面
- 按类别筛选与搜索，支持一键复制安装命令
- 包含 MCP 与社区互联索引，连接 MCP 服务器与第三方市场

## 安装方式

- Claude Code 插件市场集成，一键安装插件
- 支持手动仓库安装，适用于受控与审计环境
- 基于仓库组织的插件清单与元数据，支持自动化安装与持续更新

## 智能体与自动化工作流

- 快速部署领域专家智能体，辅助编码、审计与测试工作
- 通过命令与钩子自动化常见开发任务
- 在安全或受控环境中手动部署精选插件以满足合规与审计要求

## 社区与可扩展性

- Web UI 提供可视化浏览、过滤与示例演示以提高可发现性
- 文档中详细说明插件格式与贡献流程，注重可扩展性与可贡献性
- 作为社区发现入口，快速找到并试用第三方插件与市场集合
