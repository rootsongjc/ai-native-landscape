## 简介

Codex 是 OpenAI 推出的云端 AI 编程智能体，能够自主处理完整的软件工程任务。它可以理解代码库、生成和重构代码、编写测试以及修复跨项目的缺陷，将深度代码理解与自主执行相结合，端到端地完成复杂的工程任务。

## 主要特性

基于云端的自主执行能力，可处理包括代码生成、重构、调试和测试编写在内的完整软件工程任务。与 ChatGPT 账号或 API Key 集成，灵活访问 OpenAI 最强大的模型。兼容 MCP 协议，支持与多智能体生态系统的互操作和外部工具调用，扩展智能体能力边界。

## 使用场景

自主处理完整的软件工程任务，如构建功能特性、修复跨代码库的缺陷和生成全面的测试套件。通过自动化的代码理解和建议增强代码审查和团队知识共享。集成到开发工具链中，在现有 CI/CD 工作流内自动生成测试、文档和修复建议。

## 技术特点

通过 `~/.codex/config.toml` 进行配置，支持个性化开发环境和工作流定制。支持本地与云端混合接入模式，适配多种部署策略。开源架构，具备可扩展的插件机制和活跃的社区，持续贡献示例和改进。
