## 简介

BB Browser 将浏览器变为 AI Agent 的 API。它提供 CLI 和 MCP server，允许 AI Agent 控制浏览器同时保留用户登录状态，实现无需凭证管理的认证 Web 自动化。

## 主要特性

- AI Agent 控制浏览器的 CLI 和 MCP server。
- 保留用户登录状态实现认证自动化。
- 无需存储凭证——使用现有浏览器会话。
- MIT 协议，轻量级。

## 使用场景

- 自动化需要认证会话的 Web 任务。
- 构建 AI Agent 以登录用户身份与 Web 应用交互。
- 从 Claude Code 或其他 MCP 兼容 Agent 控制浏览器。

## 技术特点

- GitHub 5,600+ Star。
- MCP 协议兼容 Claude Code 和其他 AI Agent。
- 支持现有 Chrome 配置和 Cookie。
