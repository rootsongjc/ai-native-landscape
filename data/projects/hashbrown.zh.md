Hashbrown 是一个专为在浏览器中构建 AI 智能体而设计的框架，针对 Angular 和 React 应用量身打造。它使开发者能够将多步任务编排和外部工具调用带入前端环境，让智能体直接操控浏览器 UI。

## 主要特性

- 浏览器原生智能体执行能力，配备 Angular 和 React 专用适配层，简化与组件和路由的协作
- 内置可扩展的模型、检索和外部 API 连接器以支持常见 LLM 工作流
- 权限边界与执行隔离机制，确保在浏览器中安全调试和观测智能体行为
- 模块化架构以适配器和工作流编排为核心概念
- 使用 TypeScript 实现，与现代前端构建链和组件系统完全兼容

## 使用场景

- 增强型网页助手，在客户端协调交互逻辑和模型调用
- 自动化表单填写与数据抓取智能体，支持直接操控 UI
- 面向 UI 的任务编排器，结合页面交互与 LLM 推理
- 浏览器端 RAG 工作流入口，适用于前端驱动的检索与生成场景

## 技术特点

- 开发者可将外部模型、缓存和检索组件插入浏览器端执行路径，同时维护安全边界
- 支持模型调用、页面交互与外部服务在客户端的受控协调
- 强调与现代前端工具链和组件系统的兼容性
