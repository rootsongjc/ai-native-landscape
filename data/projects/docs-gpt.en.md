## Overview

DocsGPT is a private AI platform for agents, assistants, and enterprise search, featuring a built-in Agent Builder, deep research, and document analysis capabilities. It combines retrieval-augmented generation (RAG) with large language models to deliver citation-backed answers across multiple document formats.

## Key Features

- Wide format support including PDF, DOCX, PPTX, Markdown, HTML, and CSV with source citations.
- Multi-model and local inference support for OpenAI, Google, Anthropic, and local runtimes like Ollama.
- Built-in Agent Builder with actionable tooling, APIs, and MCP and OAuth integrations.

## Use Cases

- Enterprise document search and internal knowledge assistants with privacy-preserving self-hosting.
- Compliance investigations, legal and engineering document analysis with citation-backed evidence.
- Building private chatbots and document-driven search experiences with data under organizational control.

## Technical Details

- Python backend with React/Vite frontend, supporting both Docker and QuickStart deployment.
- MIT-licensed with an active community and a commercial cloud offering for managed deployments.
- Crawl content from URLs, sitemaps, or GitHub repositories for comprehensive document ingestion.
