Unity MCP 让 AI 助手能够通过 MCP 协议与 Unity 编辑器深度集成，实现自然语言控制、自动化场景管理、脚本编辑等多种功能。开发者可借助该工具提升工作效率，简化复杂的编辑器操作流程。

## 主要特性

- 支持自然语言指令控制 Unity 编辑器
- 提供丰富的资产、场景、脚本和菜单管理工具
- 自动化常见开发和测试流程
- 易于扩展，兼容多种 MCP 客户端

## 使用场景

- 游戏开发中的自动化资产和场景管理
- 批量脚本生成与编辑
- 提升团队协作与开发效率
- AI 驱动的 Unity 编辑器增强

## 技术特点

- 基于 Model Context Protocol (MCP) 标准
- 跨平台支持，兼容主流操作系统
- 结合 Python 服务端与 Unity 插件架构
- 开源社区活跃，持续更新
