## 简介

NotFair 提供面向 Claude Code 的开源技能集，涵盖 SEO、生成式引擎优化（GEO）、Google Ads 和 Meta Ads。营销人员可以通过 Claude Code 直接自动化广告管理、内容优化和广告效果分析。

## 主要特性

- Claude Code 的 SEO 和生成式引擎优化技能。
- Google Ads 和 Meta Ads 自动化技能。
- 内容优化和关键词分析工具。
- MIT 协议，社区维护。

## 使用场景

- 自动化 SEO 审计和内容优化工作流。
- 通过 AI 管理和优化 Google Ads 和 Meta Ads 广告系列。
- 用 Claude Code 技能构建自定义营销自动化。

## 技术特点

- GitHub 2,700+ Star。
- 专注于营销和广告自动化。
