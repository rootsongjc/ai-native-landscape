## 简介

Free LLM API Resources 是一个由社区策展维护的目录，汇总了可通过 API 访问的免费 LLM 推理资源。该仓库汇集了 OpenRouter、Vercel AI Gateway、Cloudflare Workers AI 等多家提供商和网关平台，以免费额度、速率限制、可用模型和接入链接等信息进行组织，帮助开发者快速发现和比较各种低成本方案。

## 主要特性

该目录广泛覆盖了提供免费或试用 API 访问的供应商和网关，并提供速率限制、模型可用性和直接链接等实用元数据以便快速上手。它由社区驱动，通过 Pull Request 持续更新，大多数条目指向直接的 API 端点或网关 URL，便于在实验或 CI 环境中轻松集成。

## 使用场景

该资源非常适合用于原型开发，以低成本快速验证模型能力；也可为课堂或研讨会的教学实验提供可复现的 API 参考。开发者还可以跨多个提供商比较延迟、配额和可用性，以选择最适合自身需求的方案。

## 技术特点

该仓库采用文档优先的方式，以 Markdown 清单形式组织，部分内容由脚本生成以保持可扩展性。它覆盖了 OpenRouter、Cloudflare、Vercel、Hugging Face 等多个平台，欢迎社区通过 Pull Request 贡献更新，并附有使用示例和快速入门指引。
