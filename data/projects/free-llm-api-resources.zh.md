Free LLM API Resources 是一个由社区策展维护的目录，汇总了可通过 API 访问的免费 LLM 推理资源。该仓库汇集了多家提供商和网关平台，以免费额度、速率限制、可用模型和接入链接等信息进行组织，帮助开发者快速发现和比较各种低成本方案。

## 供应商覆盖

- 汇集 OpenRouter、Vercel AI Gateway、Cloudflare Workers AI 等多家提供商
- 覆盖 Hugging Face Inference API 及其他社区托管端点
- 包含完全免费的层级和限时试用访问选项
- 大多数条目指向直接的 API 端点或网关 URL，便于集成

## 实用元数据

- 每个供应商的速率限制和配额详情
- 每个端点的可用模型及能力说明
- 各服务的上手链接和快速入门指引
- 使用示例帮助开发者快速开始

## 社区维护

- 社区驱动，通过 Pull Request 持续更新
- 文档优先方式，以 Markdown 清单形式组织
- 部分内容由脚本生成以保持可扩展性
- 欢迎贡献使用示例和新供应商条目

## 典型使用场景

- 原型开发，以低成本快速验证模型能力
- 课堂或研讨会的教学实验提供可复现的 API 参考
- 跨多个提供商比较延迟、配额和可用性
- CI/CD 环境需要免费 API 访问用于自动化测试
