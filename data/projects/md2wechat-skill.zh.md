## 简介

md2wechat-skill 是一个将 Markdown 转换为微信公众号格式文章的 CLI 工具。支持 40+ 排版样式、AI 生成封面图片和批量发布，既可作为独立 CLI 使用，也可作为 MCP 服务器集成到 AI 智能体平台。

## 主要特性

- 40+ 排版样式和微信公众号专业主题
- AI 生成文章封面图片
- 批量发布到微信公众号
- 提供 MCP 服务器，兼容 Claude Code 等智能体平台

## 使用场景

- 将 Markdown 博客文章转换为微信排版格式
- 自动化微信公众号内容发布流水线
- AI 智能体驱动的内容排版和分发

## 技术特点

- 使用 Go 构建，支持跨平台 CLI
- MCP 服务器集成，兼容 AI 智能体
- 支持 CLI 和智能体驱动的双重工作流
