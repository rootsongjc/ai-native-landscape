## Overview

Free LLM API Resources is a community-curated directory that lists free LLM inference resources accessible via API. The repository aggregates providers and gateway platforms such as OpenRouter, Vercel AI Gateway, and Cloudflare Workers AI, organizing them with notes on free quotas, rate limits, available models, and access links to help developers quickly discover and compare options.

## Key Features

The directory provides broad coverage of providers and gateways offering free or trial API access, along with practical metadata including rate limits, model availability, and direct links for quick onboarding. It is community-driven and continuously updated via pull requests, with most entries pointing to direct API endpoints or gateway URLs for easy integration.

## Use Cases

This resource is ideal for prototyping and quickly validating model capabilities at low cost, providing reproducible API references for classroom or workshop teaching exercises. It also enables developers to compare latency, quotas, and availability across multiple providers to select the best option for their needs.

## Technical Details

The repository follows a documentation-first approach organized as Markdown lists, with some parts generated by scripts for maintainability at scale. It covers platforms including OpenRouter, Cloudflare, Vercel, and Hugging Face, and welcomes community contributions through pull requests with usage examples and quickstart pointers.
