Open WebUI is a feature-rich self-hosted AI platform that supports fully offline operation. It provides a ChatGPT-like user experience, integrates various large language model runtimes (such as Ollama and OpenAI-compatible APIs), and includes a built-in RAG inference engine.

## Key Features

- Docker and Kubernetes deployment support
- Integration with Ollama and OpenAI-compatible APIs
- Complete permission and user group management
- Mobile and PWA support
- Built-in Markdown and LaTeX support
- Voice/video call functionality
- Model builder and Python function calls
- RAG and web search integration
- Image generation support
- Multi-language interface
- Plugin system support

The project is continuously maintained with regular updates and fixes.
