## 简介

Kilo Code 是一款适用于 VS Code 及其他 IDE 的开源 AI 编程助手扩展，融合了 Roo Code 和 Cline 的优秀特性并增加了创新功能。它支持通过自然语言生成代码、自动检查代码质量、执行终端命令以及实现浏览器自动化，让开发者在熟悉的开发环境中即可享受 AI 辅助编程。

## 主要特性

扩展内置了 Gemini 2.5 Pro、Claude 4 Sonnet & Opus 和 GPT-4.1 等最新 AI 模型支持，用户无需手动配置 API 密钥即可直接使用。Kilo Code 提供自然语言代码生成、智能重构、MCP 服务器市场以扩展智能体能力，以及多模式支持，可在架构师、编码者和调试器等角色间灵活切换。

## 使用场景

Kilo Code 专为寻求将 AI 结对编程集成到 IDE 工作流的开发者设计。它在通过自然语言快速原型生成、开发过程中的自动化代码质量检查，以及需要结合代码生成、终端命令和浏览器自动化的复杂多步骤任务场景中表现出色。

## 技术特点

作为 Roo Code 的分支，Kilo Code 继承了其全部功能，同时整合了来自 Cline 的 MCP 服务器市场、系统通知和简化的模型连接方式等特性，并提供更丰厚的免费额度。扩展支持自定义模式和角色，可适配不同的开发工作流和团队偏好。
