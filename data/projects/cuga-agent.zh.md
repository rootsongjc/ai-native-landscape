## 简介

CUGA 是一个面向企业的开源通用智能体框架，支持在 Web 和 API 上执行复杂任务。它集成了 OpenAPI 和模型上下文协议（MCP），提供可组合、可审计的运行时，并内置策略感知控制，适合在受管环境中部署。

## 主要特性

CUGA 支持在 Web、HTTP 和第三方 API 上执行端到端任务，采用模块化的可组合架构。它内置 OpenAPI 和 MCP 适配器以简化工具调用和能力扩展，并提供策略与权限控制机制，满足企业合规与风险管理需求。

## 使用场景

CUGA 适用于企业级智能体场景，包括自动化业务流程编排和受控的数据检索管道。它同样适用于面向客户支持的任务型智能体，以及需要策略执行和完整审计能力的自动化代理服务。

## 技术特点

CUGA 以 Python 为主实现，注重可扩展性和可观测性，提供 SDK 和运行时组件以支持渐进式企业集成。它支持多种推理模式和外部模型适配器，便于在现有企业系统中灵活引入和治理智能体功能。
