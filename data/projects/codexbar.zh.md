## 简介

CodexBar 是一款原生的 macOS 菜单栏应用程序，可让您即时查看 AI 编码工具的使用统计。它驻留在菜单栏中，一目了然地显示您的 OpenAI Codex 和 Claude Code 使用量，无需登录网页门户即可检查消耗情况。

## 主要特性

- 菜单栏中实时显示使用统计
- 同时支持 OpenAI Codex 和 Claude Code
- 无需登录——使用本地 API 访问
- 轻量级原生 macOS 界面
- 无需打开浏览器即可快速查看使用概览

## 使用场景

- 希望实时监控 AI 编码工具使用情况的开发者
- 需要跨项目追踪 OpenAI/Claude API 消耗的团队
- 偏好原生应用而非网页仪表板的用户

## 技术特点

- 使用 Swift 构建，提供原生 macOS 性能
- 菜单栏集成，随时可用
- 直接 API 集成，无需认证开销
