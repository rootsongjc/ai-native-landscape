Build with Claude is a single hub to find Claude Skills, Agents, Commands, Hooks, Plugins, and Marketplace collections to extend Claude Code, Claude Desktop, Agent SDK, and OpenClaw. It aggregates community and curated extensions into a searchable Web UI, enabling installation via the Claude plugin marketplace or manual repository-based updates.

## Plugin Discovery and Marketplace

- Rich plugin collections covering agents, commands, hooks, and skills in one searchable interface
- Filtering and search by category with one-click copyable install commands
- MCP and community indexing to find MCP servers, connectors, and third-party marketplaces

## Installation Options

- Claude Code marketplace integration for streamlined plugin installation
- Manual repository-based installation for controlled and audited deployments
- Automated installs and updates driven by repository plugin manifests and metadata

## Agent and Automation Workflows

- Rapidly deploy domain-specific agents to assist with coding, reviews, and testing
- Automate routine development tasks via commands and hooks
- Curate vetted plugins for regulated environments to ensure compliance

## Community and Extensibility

- Web UI for visual browsing, filtering, and usage examples to improve discoverability
- Documentation and contribution guides that define plugin formats and validation
- Community discovery portal for exploring third-party plugins and collections
