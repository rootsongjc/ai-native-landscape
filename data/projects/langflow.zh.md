## 简介

Langflow 是一个面向开发者和数据科学家的可视化平台，用于快速构建、测试与部署 AI 智能体与工作流。它提供画布式拖拽组件、交互式 playground、以及将流程导出为 API 或 MCP 服务的能力，使复杂的多步骤 agent 流程可以通过图形化方式设计并与现有系统集成。

## 主要特性

- 可视化画布：通过节点与连线构建数据流与模型调用链，降低上手门槛。
- 多模型与存储集成：原生支持多种 LLM、向量数据库与检索组件，便于构建 RAG 与多模态流程。
- 导出与部署：将流程导出为可部署的 API 或 MCP server，支持在生产环境中运行。
- 交互式调试与观测：内置 playground 与日志，便于调试与性能优化。

## 使用场景

- 快速原型：团队可用可视化工具快速搭建并验证 agent 设计。
- RAG 应用：结合向量库和检索组件构建文档检索增强的问答系统。
- 自动化流程：将复杂业务流程建模为 agent 工作流并对外提供 API。

## 技术特点

- 基于 Python 与前端技术栈实现，支持插件扩展与自定义组件。
- 提供详尽文档与部署指南（包括 Docker 与桌面版本）。
- 活跃社区与频繁发布，拥有大量示例、模板与 CI 流程。
