## 简介

Diffusers 是 Hugging Face 的旗舰库，提供用于图像、视频和音频生成的最先进扩散模型（基于 PyTorch）。它提供模块化的预训练模型和流水线工具箱，兼顾推理与训练需求，强调可用性和可定制性，使开发者能够轻松实验、部署和定制最新的生成式 AI 模型。

## 主要特性

Diffusers 提供开箱即用的文本到图像、图像到图像、图像修复和视频生成流水线。它具有可替换的调度器和模块化模型组件，允许用户在采样质量和速度之间精细调节。该库与 Hugging Face Hub 深度集成，可访问大量预训练检查点，并兼容多种主流硬件后端。

## 使用场景

Diffusers 非常适合在研究和创意应用中快速原型验证生成模型、构建大规模图像和媒体生成的生产推理流水线，以及使用自定义调度器和组件训练或微调扩散模型以满足特定需求。

## 技术特点

Diffusers 基于 Python 和 PyTorch 构建，采用模块化设计，流水线、调度器、模型和工具组件均可组合和扩展。它依托 Hugging Face Hub 生态系统进行模型发现与分发，支持可选的硬件特定优化，由活跃社区维护并提供丰富文档和频繁更新。
