Diffusers 是 Hugging Face 的旗舰库，提供用于图像、视频和音频生成的最先进扩散模型（基于 PyTorch）。它提供模块化的预训练模型和流水线工具箱，兼顾推理与训练需求，强调可用性和可定制性。

## 主要特性

- 开箱即用的文本到图像、图像到图像、图像修复和视频生成流水线
- 可替换的调度器和模块化模型组件，允许在采样质量和速度之间精细调节
- 与 Hugging Face Hub 深度集成，可访问大量预训练检查点
- 兼容多种主流硬件后端，支持可选的硬件特定优化
- 流水线、调度器、模型和工具组件均可独立组合和扩展

## 使用场景

- 在研究和创意应用中快速原型验证生成模型
- 构建大规模图像和媒体生成的生产推理流水线
- 使用自定义调度器和组件训练或微调扩散模型以满足特定需求
- 通过统一的易用 API 实验最新的生成式 AI 技术

## 技术特点

- 基于 Python 和 PyTorch 构建，开发者体验优先
- 依托 Hugging Face Hub 生态系统进行模型发现与分发
- 由活跃社区维护并提供丰富文档和频繁更新
