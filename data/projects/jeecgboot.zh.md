JeecgBoot 是一个 AI 驱动的低代码平台，融合了低代码与零代码双引擎能力，可大幅加速企业应用开发。平台支持一键生成完整的前后端代码，最快可在五分钟内搭建出可用的业务系统。

## 主要特性

- 内置强大的代码生成器，可根据数据库表结构一键生成全栈 CRUD 应用代码
- 集成 AI 辅助开发流程，自动推荐代码片段并减少重复编码
- 丰富的组件生态，包括安全集成、流程引擎、认证授权等模块
- 基于 Spring Boot / Spring Cloud / MyBatis-Plus 的微服务架构
- 前后端分离设计，前端使用 Ant Design Vue + Vite/TypeScript

## 使用场景

- 企业后台系统的快速搭建，通过表驱动方式快速生成数据管理与业务界面
- 项目原型与 PoC 场景，团队可快速产出可交付原型，将精力集中在业务逻辑上
- 内部工具和管理后台的快速开发，极大减少手动编码工作量
- 需要标准化架构和可复用组件的企业级 Java 项目

## 技术特点

- 基于成熟的 Java 技术栈（Spring Boot / Spring Cloud / MyBatis-Plus）构建，后端稳定可靠
- 支持通过插件、代码生成模板和流程引擎进行扩展
- 开源许可为 Apache-2.0，社区活跃且文档完善
