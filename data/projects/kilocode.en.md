## Overview

Kilo Code is an open-source AI coding assistant extension for VS Code and other IDEs that combines the best features of Roo Code and Cline while adding innovative capabilities. It enables developers to generate code through natural language, automatically check code quality, execute terminal commands, and perform browser automation, all within their preferred development environment.

## Key Features

The extension comes with built-in support for the latest AI models including Gemini 2.5 Pro, Claude 4 Sonnet and Opus, and GPT-4.1, allowing users to start immediately without manual API key configuration. Kilo Code offers natural language code generation, intelligent refactoring, an MCP server marketplace for extending agent capabilities, and multi-mode support for switching between architect, coder, and debugger roles.

## Use Cases

Kilo Code is designed for developers seeking an AI pair programmer integrated directly into their IDE workflow. It excels at rapid prototyping through natural language code generation, automated code quality checks during development, and complex multi-step tasks that require combining code generation with terminal commands and browser automation.

## Technical Details

As a fork of Roo Code, Kilo Code inherits its full functionality while integrating additional features from Cline, including an MCP server marketplace, system notifications, and simplified model connectivity with generous free credits. The extension supports custom modes and roles, making it adaptable to different development workflows and team preferences.
