## Overview

SillyTavern is an LLM frontend tailored for power users. It focuses on conversation UX and extensibility, supporting multiple model connectors and plugins. The project is open-source under AGPL-3.0 and provides local deployment options for privacy-conscious users.

## Key features

- Conversation-focused UI with advanced controls.
- Plugin architecture for integrations and extensions.
- Flexible deployment modes (local, remote, containerized).

## Use cases

- Building and testing advanced chatbots.
- Prototyping personalized assistants and role-play experiences.
- Interacting with models in offline or restricted networks.
