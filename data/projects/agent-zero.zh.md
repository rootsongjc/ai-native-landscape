## 简介

Agent Zero 是一个以提示为中心、面向扩展的开源智能体框架，强调代理间协作与可定制性。它提供持久记忆、工具化代码执行与多种模型提供器接入，旨在把计算机作为完成任务的工具。

## 主要特性

- 多代理协作与分工，支持上级/下级代理模型。
- 持久记忆与文档检索（RAG）能力，加速问题解决与知识复用。
- 可扩展的工具与仪器系统，允许用户定义自有能力和流程。
- 丰富的部署选项：Docker 化、一键启动与多提供商模型适配。

## 使用场景

- 自动化开发与运维任务，如编写脚本、部署监控或问题排查。
- 数据分析与报告自动化，结合外部工具执行复杂工作流。
- 内容创作与研究助手，汇集文档并生成摘要或计划。

## 技术特点

- 使用开放的 prompt 配置驱动行为，所有默认提示均可自定义。
- 支持 Python 与 JavaScript 工具链、浏览器代理与本地终端执行。
- 采用模块化架构，便于扩展新工具、调用外部 API 与集成第三方模型。
