## Overview

Open Design, developed by the nexu.io team, is an open-source AI design intelligence platform positioned as a local-first alternative to Anthropic's Claude Design. Following the principle of "same loop, same artifact-first mental model, none of the lock-in," it enables users to accomplish professional design work using existing coding agents in a local environment.

The platform supports automatic detection of 13+ coding agent CLIs, including Claude Code, Codex, Cursor, Gemini CLI, OpenCode, Qwen, Copilot, and more for seamless integration. Licensed under Apache 2.0, the project has seen rapid GitHub star growth with an active community.

## Key Features

- **19 Built-in Skills**: Pre-configured capabilities covering diverse design scenarios
- **71 Brand-grade Design Systems**: Ready-to-use professional design templates
- **Multi-platform Output**: Generate web, desktop, mobile prototypes, slides, images, and videos
- **HyperFrames Framework**: Advanced design framework for fine-grained control
- **Sandboxed Preview**: Isolated safe testing environment
- **Multi-format Export**: HTML/PDF/PPTX/MP4 export options
- **BYOK Mode**: Bring Your Own Key for privacy-first operation
- **Agent Auto-detection**: Automatically identifies 13+ coding agent CLIs

## Use Cases

- **Rapid Prototyping**: Generate multi-platform design prototypes using AI agents
- **Brand Design Systems**: Build brand visuals quickly based on 71 preset design systems
- **Presentation Creation**: Auto-generate professional slides exportable as PPTX
- **Cross-platform Content**: Design once, output for web, mobile, and desktop
- **Team Collaboration**: Local-first architecture ensures data privacy for enterprise use

## Technical Highlights

- Local-first architecture keeps data on-device
- Supports multiple LLM backends via BYOK
- Auto-detects locally installed coding agents
- HyperFrames-based advanced design framework
- Sandboxed preview for security
- Modular skill system supports extensibility
