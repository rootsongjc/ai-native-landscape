## 简介

Claude Ads 是一个面向 Claude Code 的付费广告审计与优化技能。它执行 250+ 自动化检查，覆盖 Google Ads 和 Meta Ads 广告系列，帮助营销人员和代理商发现问题并提升广告效果。

## 主要特性

- Google Ads 和 Meta Ads 的 250+ 自动审计检查项。
- 广告系列性能分析和优化建议。
- 直接集成到 Claude Code 工作流中。
- MIT 协议，可扩展。

## 使用场景

- 审计广告系列中的常见错误和低效问题。
- 获取 AI 驱动的广告支出优化建议。
- 在营销团队中标准化广告审核流程。

## 技术特点

- GitHub 5,400+ Star。
- 专为数字营销人员和广告代理商设计。
