## Introduction

Activepieces is an automation and workflow platform for AI agents, supporting 280+ MCP servers and a rich open-source ecosystem. It is designed for both technical and non-technical users.

## Key Features

- Intuitive interface, easy to use
- 280+ MCP servers and open-source components
- Written in TypeScript, supports hot reloading and custom development
- Human-in-the-loop, approval, form triggers
- Enterprise-grade security and self-hosting

## Use Cases

- AI agent automation and workflow integration
- Enterprise automation and data processing
- Unified management of multi-platform AI resources
- Custom AI flows and template development

## Technical Highlights

- TypeScript ecosystem, hot-reloadable components
- Supports Claude Desktop, Cursor, Windsurf, and more
- Active open-source community, frequent updates
- Multi-language and template customization
