## Overview

Hyperframes by HeyGen is an open-source tool that converts HTML into rendered video, designed specifically for AI agents. It enables programmatic video creation from code, making it possible for AI agents to generate dynamic video content at scale.

## Key Features

- HTML-to-video rendering pipeline built for AI agents.
- Programmatic video creation from structured templates.
- Designed for integration into agent workflows and automation.
- Apache 2.0 licensed by HeyGen.

## Use Cases

- Generate personalized videos at scale via AI agents.
- Create dynamic marketing and presentation videos from data.
- Build automated video production pipelines.
