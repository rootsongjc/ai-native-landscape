## Overview

CodexBar is a native macOS menu bar application that provides instant access to AI coding usage statistics. It sits in your menu bar and shows your OpenAI Codex and Claude Code usage at a glance, eliminating the need to log into web portals to check your consumption.

## Key Features

- Real-time usage statistics displayed in the menu bar
- Support for both OpenAI Codex and Claude Code
- No login required—uses local API access
- Lightweight, native macOS interface
- Quick usage overview without opening browser

## Use Cases

- Developers who want to monitor their AI coding tool usage in real-time
- Teams tracking OpenAI/Claude API consumption across projects
- Users who prefer native apps over web-based dashboards

## Technical Details

- Built with Swift for native macOS performance
- Menu bar integration for always-available access
- Direct API integration without authentication overhead
