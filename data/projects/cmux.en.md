## Overview

cmux is an open-source macOS terminal application built on top of the Ghostty terminal emulator. It rethinks the terminal UI with vertical tabs, native notifications, and first-class support for AI coding agents such as Claude Code, Codex, Gemini CLI, OpenCode, and Amp.

## Key Features

- Ghostty-based rendering for high-performance terminal output
- Vertical tab sidebar for managing many sessions at a glance
- Built-in notifications to track long-running agent tasks and builds
- Designed for programmability and automation of AI coding agent workflows
- tmux-style multiplexing and session management

## Use Cases

- Running multiple AI coding agents in parallel sessions
- Monitoring long-running agent tasks through native notifications
- Power-user terminal workflows on macOS with a modern interface

## Technical Details

- Native macOS application built on the Ghostty core
- Integrates with Claude Code, Codex, Gemini, OpenCode, and Amp
- Licensed under a custom non-standard license (see repository LICENSE)
