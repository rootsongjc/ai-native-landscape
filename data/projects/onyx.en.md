## Overview

Onyx is a self-hostable, enterprise-ready chat UI that works with any LLM. It includes Agents, web search, hybrid RAG, deep research tools and connectors to over 40 knowledge sources, enabling secure, scalable knowledge retrieval and agentic workflows.

## Key Features

- Deployable via Docker, Kubernetes or Terraform with quickstart guides and cloud provider instructions.
- Advanced RAG pipeline and indexing with connectors and document permissioning.
- Agent actions and MCP support to let agents interact with external systems and automate tasks.

## Use Cases

- Enterprise knowledge search and Q&A with fine-grained access control.
- Offline or airgapped deployments for secure environments.
- Building custom agents, deep research assistants and multi-source conversational interfaces.

## Technical Highlights

- Compatible with hosted and self-hosted LLMs (OpenAI, Anthropic, Gemini, Ollama, etc.).
- Includes SDKs, CLI and management UI for integration and extensibility.
- Enterprise features: SSO (OIDC/SAML), RBAC, credential encryption and audit logs.
