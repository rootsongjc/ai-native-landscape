## 简介

Agentic Inbox 是 Cloudflare 构建的自托管邮件客户端，集成 AI Agent 自动阅读、摘要和处理邮件。完全运行在 Cloudflare Workers 上，无需外部依赖，展示了 AI Agent 如何管理通信工作流。

## 主要特性

- AI Agent 自动阅读、摘要和回复邮件。
- 完全运行在 Cloudflare Workers，无需外部服务器。
- 自托管，完全隐私控制。
- Cloudflare 官方出品的 Agent 应用参考架构。

## 使用场景

- 用 AI Agent 自动化邮件分类和回复。
- 在 Cloudflare 边缘网络上构建自定义邮件工作流。
- 自托管私有 AI 邮件助手。

- GitHub 3,800+ Star。
- Cloudflare 官方项目，Apache 2.0 协议。
- 完全运行在边缘计算上，零冷启动。
