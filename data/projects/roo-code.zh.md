## 简介

Roo Code 是一个 AI 驱动的自主编码代理，直接在代码编辑器中运行，充当由专业化 AI 代理组成的完整开发团队。它支持自然语言交互，可读写工作空间文件、执行终端命令和自动化浏览器操作，端到端地处理复杂开发任务。

## 主要特性

- 自然语言交互，支持在整个项目工作空间中读取、编写和重构代码。
- 直接执行终端命令和自动化浏览器操作，适用于测试和 UI 自动化工作流。
- 兼容任何 OpenAI 兼容 API 或自定义模型，包括本地和自托管端点。
- 自定义模式功能，用户可定义架构师、QA 工程师、产品经理等专业化 AI 角色。

## 使用场景

开发者使用 Roo Code 完成从全新项目搭建到遗留代码维护的各种任务，利用其深度上下文理解能力导航复杂代码库。团队通过自定义模式获得角色特定的 AI 辅助，个人开发者则加速代码生成、调试、测试编写和文档编写等日常工作流。

## 技术特点

Roo Code 深度集成到 VS Code 和其他主流编辑器中，无需在工具之间切换。它支持具有持久项目上下文的多轮对话，分析代码结构和依赖关系图，并内置安全机制，在执行删除文件或系统命令等敏感操作前提示用户确认。
