## 简介

browser-use 是一个用 Python 构建的库，旨在让 AI 能够控制浏览器，支持 Playwright、WebRTC 等后端并提供云 demo 服务，用于表单填写、网页抓取与端到端自动化 agent 场景。

## 主要特性

- 简易的 Agent API，可与 LLM 联动实现网页操作
- 丰富的示例与云端托管（cloud.browser-use.com）便于快速验证
- 支持多地域文档以及本地/云部署模式

## 使用场景

- 自动化网页操作（表单、爬取、测试）
- 为 LLM 提供浏览器工具能力，实现 RAG 或工具调用
- 教学、hackathon 与原型验证场景

## 技术特点

- 以 Playwright 为基础的稳定浏览器控制能力
- 支持 MCP 集成与多语言文档
- MIT 许可，社区活跃、更新频繁
