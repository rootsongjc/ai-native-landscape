## Overview

Siyuan is a privacy-first, self-hosted PKM tool that combines rich note-taking, full-text search, OCR integrations, and import/export capabilities. It supports both desktop and web access, making it suitable for users and teams who require local data control and robust knowledge organization.

## Key features

- Self-hosting and privacy control.
- Block-level linking and powerful full-text search.
- Extensible via plugins and integrations.

## Use cases

- Personal knowledge management and archival.
- Team documentation and internal knowledge bases.

## Technical notes

- Built with TypeScript and Go for responsive UI and stable backend.
