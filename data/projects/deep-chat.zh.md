## 简介

Deep Chat 是一个可嵌入网页的一行代码 AI 聊天组件，支持直接连接 OpenAI、HuggingFace、Cohere 等 API，也能在浏览器端托管轻量模型。它提供丰富的 UI 功能（头像、文件传输、摄像头与麦克风支持、语音输入/输出）与可扩展的拦截器/连接器机制。

## 主要特性

- 一行代码嵌入：支持多框架（React / Vue / Svelte / Angular 等）。
- 多通道输入输出：支持文件、摄像头拍照、麦克风录音、文本与语音交互。
- 直接连接与拦截器：支持直接调用第三方 API 或通过代理服务安全接入。
- 浏览器端模型：可在客户端运行轻量模型以减少后端依赖。

## 使用场景

- 网站快速集成聊天机器人或帮助中心界面。
- 本地/离线演示场景，使用浏览器端模型进行原型验证。
- 需要语音交互（STT / TTS）与文件上传的智能客服或演示页面。

## 技术特点

- 使用 TypeScript 编写并提供多个前端适配包与示例服务器模板。
- 丰富的文档与 Playground，支持直接在页面配置与调试组件行为。
- MIT 许可证，社区活跃，支持自部署与定制化扩展。
