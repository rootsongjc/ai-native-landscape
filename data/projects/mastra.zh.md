## 简介

Mastra 是一个用 TypeScript 构建的 AI agent 框架，支持 agents、workflows、RAG 与多种 LLM 提供商，便于在服务器或无服务器环境中部署助手与自动化工作流。

## 主要特性

- 提供统一的模型路由与 SDK，支持多家 LLM 提供商
- 支持有状态的工作流（Durable workflows）与可观测性（OpenTelemetry）
- 丰富的工具、集成与示例，方便快速构建生产级 agents

## 使用场景

- 构建客服、会议助手、语音交互等生产级 agent
- 将现有内部工具通过 agents 自动化编排
- 在无服务器环境中部署轻量级助手

## 技术特点

- 基于 TypeScript，提供强类型工具与 SDK
- 内置 RAG 管道、Evals 测试框架与可插拔集成
- 丰富的 CI/CD 与测试支持，适合团队协作开发
