## 简介

AIPex 是一款开源的浏览器自动化扩展，旨在通过自然语言指令让用户无需编写代码即可执行复杂的多步骤浏览器任务。它兼容主流浏览器，支持多标签、多窗口协同工作，并提供智能的数据抽取、表单自动化与页面交互能力，适合日常办公自动化、研究采集与数据监控等场景。

## 主要特性

- 自然语言控制：使用类人语言下达指令，由系统解析并执行对应浏览器操作；
- 多步骤工作流：支持组合与复用自动化步骤，完成复杂任务链；
- 智能元素识别：基于视觉与语义的元素定位，适应动态页面布局；
- 数据抽取与整理：自动抓取页面结构化信息并导出；
- 开发者友好：提供开放源码与插件机制，便于扩展与二次开发。

## 使用场景

- 日常办公自动化：自动填写表单、批量下载与数据整理；
- 研究与爬取：跨站点信息汇总、价格监控与舆情采集；
- 测试与回归：记录并回放用户操作用于前端自动化测试；
- 协同工作流：在多窗口、多标签间协调复杂任务与数据流转。

## 技术特点

- 基于 TypeScript + React 构建，插件化架构便于扩展；
- 使用上下文感知的解析器支持 MCP 风格的工具集成；
- 支持本地与云端运行模式，优化性能与隐私控制；
- 遵循 MIT 许可，代码与开发文档在 GitHub 上公开维护。
