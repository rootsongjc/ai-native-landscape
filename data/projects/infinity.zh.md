## 简介

Infinity 是一个 AI 原生推理引擎，专为高性能嵌入、重排序和分类工作负载而构建。它为检索增强生成和搜索应用中最常用的 AI 模型类型提供低延迟、高吞吐的推理服务，通过统一的 API 支持广泛的嵌入模型和重排序架构。

## 主要特性

Infinity 为流行的嵌入模型和重排序器提供优化的推理服务，支持包括稠密向量、稀疏向量和结构化字段在内的多种数据类型。它提供开发者友好的 Python SDK 和单二进制部署方式，以及为高 QPS 生产工作负载设计的内置可观测性和基准测试工具。

## 使用场景

Infinity 非常适合驱动向量搜索和检索增强生成（RAG）系统、构建相似度推荐引擎，以及大规模部署分类模型。企业可以将 Infinity 私有化部署以满足合规性要求，并通过 Python SDK 快速将嵌入和重排序推理集成到 LLM 应用中。

## 技术特点

Infinity 通过混合索引架构实现毫秒级查询延迟和高 QPS 吞吐量，统一向量、稀疏和全文索引。它支持作为独立二进制运行或嵌入 Python 进程进行灵活部署，采用 Apache-2.0 许可证，方便开源社区和企业采纳与扩展。
