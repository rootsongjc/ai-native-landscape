## Overview

md2wechat-skill is a CLI tool that converts Markdown to formatted WeChat public account articles. It supports 40+ formatting styles, AI-generated cover images, and batch publishing, designed as both a standalone CLI and an MCP server for integration with AI agents.

## Key Features

- 40+ formatting styles and professional themes for WeChat articles
- AI-generated cover images for articles
- Batch publishing to WeChat public accounts
- Available as MCP server for Claude Code and other agent platforms

## Use Cases

- Converting Markdown blog posts to WeChat-formatted articles
- Automated WeChat public account content publishing pipeline
- AI agent-driven content formatting and distribution

## Technical Details

- Built with Go for cross-platform CLI support
- MCP server integration for AI agent compatibility
- Supports both CLI and agent-driven workflows
