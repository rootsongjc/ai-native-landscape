Bun 是一个极速的 JavaScript 运行时、打包器、测试运行器和包管理器——四合一的高性能平台。它以单一二进制文件分发，注重启动速度、依赖安装与构建性能，被广泛用作 AI 工具链和开发流程的高性能运行时。

## 一体化工具链

- 运行时、打包、包管理和测试功能合一，减少工具链复杂度
- 无需额外安装 webpack、jest 或 npm/yarn 等独立工具
- 单一二进制文件分发，跨平台安装与部署极为便捷

## 性能表现

- 优化的 I/O 与启动流程，显著缩短脚本执行与构建时间
- 内置高性能 JavaScript 引擎与原生网络 I/O，提升并发性能
- 原生包管理与打包流程，支持常见 npm 包的快速安装和解析

## 兼容性

- 与 Node.js 常见 API 兼容，生态兼容性广泛
- 对现代 ECMAScript 特性有良好支持，包括最新语言提案
- 为现有 Node.js 项目和脚本提供无缝迁移路径

## AI 与边缘负载

- 广泛用作 AI 工具链和开发的高性能运行时
- 适合在边缘或无服务器环境中运行轻量服务
- 快速启动特性使其成为需要冷启动优化的微服务与边缘函数的理想选择
