## 简介

Chaterm 是一个面向云和基础设施管理的开源 AI 终端。它支持团队通过自然语言和智能智能体来部署、排障和自动化服务，内置 SSH 连接管理、命令确认和 MCP 协议支持。

## 主要特性

- AI Agent 支持 Command 模式和 Agent 模式进行基础设施运维
- SSH 连接管理，带命令确认和安全边界
- MCP 协议支持，可连接外部 AI 工具和工作流
- 自然语言到命令的生成，配合执行治理
- 面向团队运维设计，支持基于角色的命令策略

## 使用场景

- 团队级服务器运维，AI 辅助排障
- 通过智能体驱动的命令生成标准化基础设施工作流
- 减少重复性 DevOps 任务，同时保持安全护栏

## 技术特点

- 支持 Agent 和 Command 模式，具备不同权限级别
- 集成 MCP 用于外部工具编排
- 关联 Terminal-Bench、CNCF Landscape 和 OpenSSF Scorecard 等生态指标
