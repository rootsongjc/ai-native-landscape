## 简介

Langfuse 是面向 LLM 应用开发的开源平台，支持团队协作、监控、评估和调试。用户可自建或使用云服务，快速集成主流 LLM 框架，提升 AI 应用的可观测性和开发效率。

## 主要特性

- LLM 调用链路追踪与日志分析
- 支持提示词管理与版本控制
- 多维度评测与数据集管理
- 丰富的 API 与 SDK 集成
- 可视化 Playground 与实时调试

## 使用场景

- 企业级 LLM 应用开发与运维
- 多团队协作的 AI 项目管理
- 复杂 AI 工作流的监控与优化
- 评测与持续改进 LLM 产品

## 技术特点

- 支持自托管与云部署，易于集成
- 开源架构，兼容主流 LLM 框架
- 完善的 API 与 OpenTelemetry 集成
- 高性能数据采集与分析能力
