## Overview

Netcatty is an AI-powered SSH workspace that combines terminal sessions, SFTP file browser, and an AI assistant called Catty Agent. Built with Electron and xterm.js, it provides a unified workspace for managing multiple remote servers with AI-assisted diagnostics, command generation, and multi-host operations.

## Key Features

- Integrated SSH terminal, SFTP browser, and session management in one workspace
- Catty Agent for AI-powered server diagnostics, log analysis, and command generation
- Multi-host support with split-screen terminals and vault-based host management
- Cross-platform support for macOS, Windows, and Linux
- AI-assisted multi-host operations with confirmation workflows

## Use Cases

- Managing multiple remote servers with a unified SSH/SFTP workspace
- AI-assisted log analysis and server diagnostics across hosts
- Reducing context switching between terminal, file transfer, and host management

## Technical Details

- Built with Electron, React, Radix UI, and xterm.js
- Vault-based host management for organizing server connections
- Catty Agent understands remote server context for contextual AI assistance
