## 简介

DocsGPT 是一个面向企业与研究的私有 AI 平台，内置 Agent Builder、深度研究与文档分析功能，支持智能体、助手与企业搜索场景。它结合检索增强生成（RAG）与大语言模型，跨多种文档格式提供带来源引用的可靠答案。

## 主要特性

- 多格式文档解析：PDF、DOCX、PPTX、Markdown、HTML、CSV 等，附带证据引用以减少幻觉。
- 多模型与本地推理：兼容 OpenAI/Google/Anthropic，亦支持本地模型（如 Ollama）。
- 内置 Agent Builder 与可执行工具 API，支持 MCP 与 OAuth 安全集成。

## 使用场景

- 企业文档检索、内部知识库问答与合规审计支持，数据可完全自托管。
- 法务与研发文档搜索，提供带引用的证据展示以提升可审计性。
- 构建私有客服、文档驱动的智能搜索或内部智能体平台。

## 技术特点

- 后端以 Python 为主，前端使用 React/Vite，支持 Docker 与 QuickStart 快速部署。
- 可从网址、站点地图或 GitHub 仓库抓取内容，实现全面的文档摄取。
- MIT 许可证，社区活跃并提供付费云版本与企业支持。
