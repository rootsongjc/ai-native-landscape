## Overview

Last30Days is an AI agent skill for Claude Code that performs multi-platform research across Reddit, X (Twitter), YouTube, Hacker News, Polymarket, Bluesky, TikTok, Instagram, and the open web. It synthesizes findings into a grounded summary with source citations, enabling deep research on any topic.

## Key Features

- Searches across 9+ platforms including Reddit, X, YouTube, Hacker News, and Polymarket
- Synthesizes multi-source findings into structured summaries with citations
- Recency-focused research with configurable time windows
- Installable as a Claude Code skill for seamless agent integration
- Supports trend analysis and topic tracking across social media

## Use Cases

- Research trending topics and conversations across social platforms
- Get a synthesized briefing on any subject with cited sources
- Track emerging trends and discussions across Reddit, X, and HN

## Technical Details

- Designed as a Claude Code skill following the agent skill specification
- Supports multiple search backends and social media APIs
- Outputs structured markdown with source attribution
