## 详细介绍

Pi Monorepo 是一个大型的 AI agent 工具箱，包含统一的 LLM 接口、多种编码智能体（coding agent）、终端 UI（TUI）、Web UI 组件及部署工具包，方便在不同场景中构建与运行智能体应用。

## 主要特性

- 多包 monorepo：多个独立可发布的包（CLI、SDK、Web 组件）。
- 统一 LLM 适配层，支持多个模型提供商。
- 强调开发者体验：CLI、终端 UI 与示例应用。

## 使用场景

- 构建定制化编码智能体（code agent）。
- 在终端或 Web 中嵌入 AI 聊天/协作功能。
- 管理 vLLM 部署与多模型路由。

## 技术特点

- TypeScript 主导，现代 monorepo 架构。
- 提供丰富的示例与工具链（构建、测试与 CI）。
