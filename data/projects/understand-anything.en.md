## Overview

Understand Anything transforms codebases into interactive knowledge graphs that developers can explore, search, and query using natural language. It integrates directly with popular AI coding tools including Claude Code, Codex, Cursor, Copilot, and Gemini CLI to provide deep code understanding.

## Key Features

- Converts code into explorable interactive knowledge graphs
- Natural language search and Q&A over codebases
- Native integration with Claude Code, Codex, Cursor, Copilot, and Gemini CLI
- Supports codebase analysis and knowledge base generation
- Works as a skill for multiple AI coding agents

## Use Cases

- Explore and understand unfamiliar codebases through visual knowledge graphs
- Ask questions about code architecture and dependencies in natural language
- Generate persistent knowledge bases for team onboarding and documentation

## Technical Details

- Built in TypeScript with support for multiple AI agent platforms
- Uses graph-based code analysis to map relationships between code entities
- Provides skills/plugins for major AI coding tools
